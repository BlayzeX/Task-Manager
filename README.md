# Task Manager ✅

A gamified task manager built with HTML, CSS & Vanilla JavaScript.
Complete your tasks, earn XP, and level up!

## Features
- Add & delete tasks
- Complete tasks → earn XP ⚡
- XP progress bar with level up system
- Tasks saved after page refresh (localStorage)
- Peaceful wheat field aesthetic 🌾

## Built With
- HTML
- CSS
- Vanilla JavaScript

![Task Manager Preview](preview.png)

## How It Works
- Add a task → type and press Enter or click +
- Complete a task → check the checkbox → earn +50 XP
- Reach 500 XP → Level Up! 🎉
- Delete a task → click X
- All data is saved automatically
