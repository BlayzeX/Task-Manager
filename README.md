# Task Manager ✅

A gamified task manager built with HTML, CSS & Vanilla JavaScript.
Manage your daily tasks and earn XP as you complete them.

## Features
- Add & delete tasks
- Complete tasks → earn +50 XP
- XP progress bar
- Level up system (every 500 XP)
- Peaceful wheat field aesthetic 🌾

## Built With
- HTML
- CSS
- Vanilla JavaScript
